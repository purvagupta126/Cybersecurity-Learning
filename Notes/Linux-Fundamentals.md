Linux Fundamentals Notes

Why I learned this:

Many cybersecurity tools and servers use Linux, so I started with Linux basics.

Commands I learned:

pwd

* Shows the current directory.

ls

* Lists files and folders.

cd

* Changes directory.

cat

* Displays file contents.

mkdir

* Creates a new directory.

touch

* Creates a new file.

cp

* Copies files.

mv

* Moves or renames files.

rm

* Deletes files.

Things I found interesting:

* Linux is widely used in cybersecurity.
* Many security tools run on Linux.
* The command line gives powerful control over a system.
